# 兔兔小小科學院

給 3 歲半幼兒使用的單頁互動學習網站。所有功能都在瀏覽器本機執行，不需要帳號，也不會上傳孩子的學習紀錄。

## 學習世界

- 今天：先用四個選項告訴兔兔現在的感覺，再進入「暖身複習 → 新發現 → 生活任務」，完成後培養五階段兔兔花園。
- ABC：看字母、大小寫配對、聽音選字母與熟悉度進度。
- 兔兔故事：人物與地點、因果、情緒轉變、故事順序、幫助朋友、預測下一步。
- 兔兔心情：開心分享、生氣停一下、難過說需要、害怕找大人、等待輪流與身體界線。
- 寶寶數學：1–5 點數、一眼看數量、多與少、形狀、規律、位置詞。
- 寶寶物理：斜坡滾球、上與下、浮與沉、推力、聲音震動、光與影子。
- 寶寶量子：光的一小包、機率位置、雙路徑明暗條紋、光子能量、量測、穿隧。
- 寶寶科學：顏色混合、磁鐵、植物向光、融化、溶解、風與風車。

網站目前共有 36 個引導活動：6 個故事、6 個心情情境、6 個數學與 18 個科學活動。數學與科學採用「先猜、再試、看結果」；故事採用「看線索、回答第一題、回答追問、說給兔兔聽」；心情屋採用「辨認感受、選擇安全做法、練習一句話」。選擇不同不扣星星，每題只有第一次完成會獲得星星。

故事屋與心情屋都不使用麥克風，也不評分孩子說話。每篇完成後只提供一句可選擇練習的短句，讓孩子用動作、單字或短句與照顧者互動。心情屋不做發展診斷、不羞辱、不強迫分享，並在身體界線情境示範說停、離開和找可信任的大人。量子內容是幼兒概念模型，家長提示中保留必要的科學邊界，避免把示意動畫當成肉眼可見的真實粒子軌跡。

首頁的「兔兔今天陪你」提供開心、想慢慢來、有點難過、需要抱抱四個當下選項。它沒有正解、不加星、不計入每日任務，也不要求孩子說明原因；選擇只保存在當天的本機進度中，可隨時重選，隔日不會沿用成固定標籤。

「今天」會先讓孩子選擇當下狀態，再優先複習目前較不熟的字母，接著依日期在故事、數學、心情、物理與科學之間輪替尚未完成的內容，量子作為好奇心延伸，最後安排一個不需上傳照片或錄音的生活探索。當天的新發現會固定為同一篇，完成與重玩不會突然換題；隔天才重新選擇。每日三項任務各自只記一次成長，舊版儲存在 `bunnyAlphabetMemoryV1` 的進度可直接沿用。

介面針對 iPad mini 直向與橫向調整，主要觸控按鈕至少 44 CSS px，並支援觸控裝置較大的操作區。中文語音 rate 為 0.84，英文語音 rate 為 0.82。

## 使用方式

直接開啟 `index.html`，或使用 GitHub Pages 網址。建議使用 iPad mini 或較大的觸控畫面，也可在 Safari、Chrome 或 Edge 的近期版本使用；語音音色會依裝置已安裝的系統語音而不同。

長期內容規劃與分齡解鎖原則請見 [`CURRICULUM_ROADMAP.md`](CURRICULUM_ROADMAP.md)。
