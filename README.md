# 兔兔小小科學院

給 3 歲半幼兒使用的單頁互動學習網站。所有功能都在瀏覽器本機執行，不需要帳號，也不會上傳孩子的學習紀錄。

## 學習世界

- 今天：自動安排「暖身複習 → 新發現 → 生活任務」，完成後培養五階段兔兔花園。
- ABC：看字母、大小寫配對、聽音選字母與熟悉度進度。
- 寶寶數學：1–5 點數、一眼看數量、多與少、形狀、規律、位置詞。
- 寶寶物理：斜坡滾球、上與下、浮與沉、推力、聲音震動、光與影子。
- 寶寶量子：光的一小包、機率位置、雙路徑明暗條紋、光子能量、量測、穿隧。
- 寶寶科學：顏色混合、磁鐵、植物向光、融化、溶解、風與風車。

6 個數學活動與 18 個科學活動都採用「先猜、再試、看結果」流程，猜想不同也不扣星星。每題都有可重播語音、動畫、第一次完成的星星獎勵與家長說明。量子內容是幼兒概念模型，家長提示中保留必要的科學邊界，避免把示意動畫當成肉眼可見的真實粒子軌跡。

「今天」會優先複習目前較不熟的字母，再帶到尚未完成的數學／科學活動，最後安排一個不需上傳照片或錄音的生活探索。每日三項任務各自只記一次成長，重玩不會重複增加星星。舊版儲存在 `bunnyAlphabetMemoryV1` 的進度可直接沿用。

介面針對 iPad mini 直向與橫向調整，主要觸控按鈕至少 44 CSS px，並支援觸控裝置較大的操作區。中文語音 rate 為 0.84，英文語音 rate 為 0.82。

## 使用方式

直接開啟 `index.html`，或使用 GitHub Pages 網址。建議使用 iPad mini 或較大的觸控畫面，也可在 Safari、Chrome 或 Edge 的近期版本使用；語音音色會依裝置已安裝的系統語音而不同。

長期內容規劃與分齡解鎖原則請見 [`CURRICULUM_ROADMAP.md`](CURRICULUM_ROADMAP.md)。
