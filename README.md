# 兔兔小小科學院

給 3 歲半幼兒使用的單頁互動學習網站。所有功能都在瀏覽器本機執行，不需要帳號，也不會上傳孩子的學習紀錄。

## 學習世界

- ABC：看字母、大小寫配對、聽音選字母與熟悉度進度。
- 寶寶物理：斜坡滾球、上與下、浮與沉。
- 寶寶量子：光的一小包、機率位置、雙路徑明暗條紋。
- 寶寶科學：顏色混合、磁鐵、植物向光生長。

每個科學主題都有可重播語音、按鈕動畫、星星獎勵與家長說明。量子內容是幼兒概念模型，家長提示中保留必要的科學邊界，避免把示意動畫當成肉眼可見的真實粒子軌跡。

## 使用方式

直接開啟 `index.html`，或使用 GitHub Pages 網址。建議在 Safari、Chrome 或 Edge 的近期版本使用，語音音色會依裝置已安裝的系統語音而不同。

長期內容規劃與分齡解鎖原則請見 [`CURRICULUM_ROADMAP.md`](CURRICULUM_ROADMAP.md)。

## 檔案

- `index.html`：GitHub Pages 入口。
- `bunny_alphabet_memory.html`：保留的本機入口。
- `bunny_*_3d_v1.webp`：網站載入的高品質 3D 場景。
- `CURRICULUM_ROADMAP.md`：3.5–8 歲陪伴成長課程與產品路線圖。

PNG 母檔保留在本機 workspace，不納入公開 repository。
