# 兔兔小小科學院

給 3 歲半幼兒使用的兔兔互動學習網站。所有功能都在瀏覽器本機執行，不需要帳號，也不會上傳孩子的學習紀錄。

## 學習世界

首頁固定只保留「今天、3D 玩、ABC、世界地圖」四個大入口，不需要在一長排選單中左右滑動。世界地圖再把 11 個既有目的地分成「動手玩 3D、故事與心情、小小學科、語言」四區，並直接顯示各區的本機進度。

- 今天：先用四個選項告訴兔兔現在的感覺，再進入「暖身複習 → 新發現 → 生活任務」，完成後培養五階段兔兔花園。
- 3D 遊樂場：直接碰紅色球、找比較大的球，再碰斜坡上的球看它往下滾；不需要拖曳、旋轉視角或閱讀規則。
- 3D 數學：一根一根摘胡蘿蔔、找出指定數量、比較哪邊較多，再把一根放進籃子看見「加一」。
- 3D 科學：依序探索重力與運動、浮沉與材料、光影與小光點；每一幕只需碰唯一發光的場景物件，不先考答案。
- 3D 生活：跟兔兔依序練洗手、收玩具與安心出門；每一步只碰唯一發光的生活物件，完成後再和大人到真實生活試一次。
- ABC：看字母、大小寫配對、聽音選字母與熟悉度進度。
- 兔兔故事：人物與地點、因果、情緒轉變、故事順序、幫助朋友、預測下一步。
- 兔兔心情：開心分享、生氣停一下、難過說需要、害怕找大人、等待輪流與身體界線。
- 寶寶數學：1–5 點數、一眼看數量、多與少、形狀、規律、位置詞。
- 寶寶物理：斜坡滾球、上與下、浮與沉、推力、聲音震動、光與影子。
- 寶寶量子：光的一小包、機率位置、雙路徑明暗條紋、光子能量、量測、穿隧。
- 寶寶科學：顏色混合、磁鐵、植物向光、融化、溶解、風與風車。

網站目前共有 36 個平面引導活動，另有四個 Three.js 3D 世界：6 個故事、6 個心情情境、6 個數學與 18 個科學活動，加上 3D 遊樂場、3D 數學花園、3D 科學實驗室與 3D 生活小屋。所有平面活動都改成「碰選項就立即看結果」，不再要求先選答案、再尋找另一個執行按鈕；故事與心情題第一步後只出現一個明確的「下一步」。選擇不同不扣星星，每題只有第一次完成會獲得星星。

自由探索時，孩子可從世界地圖選目的地；「今天」任務開始後會暫時收起四個入口，只留下目前任務與固定返回操作，完成後才恢復導覽。這個切換只改變畫面層級，不會更動 `bunnyAlphabetMemoryV1` 的舊進度、星星或解鎖規則。

3D 遊樂場使用程序化兔兔、彩球、斜坡、光影和待機動畫。三個任務都以 raycasting 接收真實 canvas 觸控，完成後每天最多增加一次星星；它是顏色、大小與斜坡因果的短遊戲，不計入原本 36 題的探索總數。Three.js 隨網站保存在本機，不依賴 CDN；WebGL 無法使用時會顯示圖卡替代內容。

3D 數學花園同樣使用真實 canvas 觸控，每回合有四個短任務：摘指定數量、找指定數量、比較多少與加一。第一階只出現 1–3；兩個完整回合的兩次選擇都首次答對後，才依序放大到 1–4、1–5。選錯只會陪孩子重新數，不扣星、不降級，也不顯示失敗。每個完成回合會保存在本機供漸進使用，但每天最多增加一顆星；當每日「新發現」輪到數學時，完成 3D 回合即可完成該任務，不重複獎勵。

3D 科學實驗室取消答題式互動，改成「兔兔邀請觀察 → 孩子碰發光物件 → 場景呈現結果」的短循環。三條路線各有三幕：小球掉落、斜坡滾球與推動小車；小船浮起、石頭下沉與冰融化；手電筒影子、鏡面反射與光點抵達探測器。錯碰背景只會再次突出目前目標，沒有失敗或扣分。三條路線依序輪替，完成回合與看過的概念只保存在本機，同一天最多增加一顆星。

3D 生活小屋把原本只靠家長閱讀的生活任務改成孩子可先看、先碰的動畫示範。洗手泡泡有用水、肥皂、沖洗與擦乾四步；玩具回家與安心出門各有三步，共 10 個生活動作。畫面不使用拖曳、鏡頭控制、倒數或失敗提示，錯碰只會再次突出目前物件。每回合會保留看過的步驟與主題次數，但同一天最多增加一顆星；每日第三件任務會固定進入當天的生活路線，完成後才算花園任務完成。

光影路線的最後一幕是 photon detection 的幼兒概念模型：亮點代表一次探測事件，不代表肉眼能看見單一光子在空中飛行。家長小站與課程地圖都保留這項科學邊界。

故事屋與心情屋都不使用麥克風，也不評分孩子說話。每篇完成後只提供一句可選擇練習的短句，讓孩子用動作、單字或短句與照顧者互動。心情屋不做發展診斷、不羞辱、不強迫分享，並在身體界線情境示範說停、離開和找可信任的大人。量子內容是幼兒概念模型，家長提示中保留必要的科學邊界，避免把示意動畫當成肉眼可見的真實粒子軌跡。

首頁的「兔兔今天陪你」提供開心、想慢慢來、有點難過、需要抱抱四個當下選項。它沒有正解、不加星、不計入每日任務，也不要求孩子說明原因；選擇只保存在當天的本機進度中，可隨時重選，隔日不會沿用成固定標籤。

「今天」可先選擇當下狀態，也可直接按「開始第一件」。網站會優先複習目前較不熟的字母，接著依日期在故事、數學、心情、物理與科學之間輪替尚未完成的內容；輪到數學時會直接進入 3D 數學花園，輪到物理、科學或量子時則分別進入 3D 實驗室的運動、浮沉或光路線，最後進入當天固定的 3D 生活路線，再由成人陪同到真實生活練一次。當天的新發現與生活路線會固定，完成與重玩不會突然換題；隔天才重新選擇。每日三項任務各自只記一次成長，舊版儲存在 `bunnyAlphabetMemoryV1` 的進度可直接沿用。

## 幼兒操作導引

- 平常只需要認得四個入口；更多內容集中在世界地圖，不需要橫向滑動尋找。
- 首頁的一鍵開始會自動選擇第一個尚未完成的任務，不必先尋找三張任務卡。
- 從首頁進入每日任務時會開啟專注模式，暫時收起四個主入口、不相關的模式與選題，讓畫面只保留當下要做的事情；完成後導覽會重新出現。
- 故事、心情、數學與科學題只需碰一個選項就會立即動起來；需要第二輪的故事與心情題只會再顯示一個「下一步」。
- 3D 遊樂場每次只說一個動作，錯碰不扣星，只會溫和提示再找找。
- 3D 數學每次只突出目前可碰的胡蘿蔔或籃子；點數採逐物件觸控，選錯會讓兩邊物件慢慢跳起來協助重數。
- 3D 科學每一幕只有一個發光目標；碰到非目標區只會再次閃亮提示，結果停留後才自動進入下一幕。
- 3D 生活每次只呈現一個步驟與一個發光物件；完成後用「先、接著、最後」回顧，再請成人陪同到生活中練習。
- 點選學習世界、ABC 模式或下一題後，畫面會直接定位到可操作內容；程式初始載入不會自行跳動。
- 所有學習頁都有固定的「回今天」。每日任務完成後會明確顯示「完成，回今天」；故事、心情、數學與科學活動完成後也可直接前往下一題。
- 回到自由探索後，完整模式與六題選單會重新出現，不會限制家長主動選題。

## 家長小站

「今天」頁底部提供預設收合的家長小站，孩子進行三件任務時不會被額外資訊打擾。展開後可查看：

- 今天完成幾件任務，以及最近 7 天有任務紀錄的陪伴日與任務數。
- 36 個活動的已探索總數，以及故事、心情、數學、物理、量子、科學各自的 0–6 探索地圖。
- 依現有字母熟悉度整理的 3 個較少練到字母；這不是能力判定或發展診斷。
- 3D 數學目前提供的數量範圍、完成回合與練習概念；這只描述網站內容，不是能力等級。
- 3D 科學三條路線的完成回合、最近與下一個探索主題；這只是使用紀錄，不是科學能力或發展診斷。
- 3D 生活三條路線的完成回合、最近與下一個練習主題；這只代表看過網站示範，不代表孩子能獨立完成。
- 今天生活任務的 3 分鐘成人陪玩方式、材料安全提醒與練習重點。
- 兔兔聲音試聽室：分別比較裝置上的中文／英文音色、保留自動選擇或固定偏好的音色，並切換「柔和／再慢一點」兩段語速。
- 自然語音／只要鈴聲切換，以及需要再次確認的全部進度重置；只要鈴聲模式仍可由成人主動試聽音色，不會改動孩子的播放設定。

摘要只讀取既有 `bunnyAlphabetMemoryV1` 本機資料，不建立帳號、不做排名或同齡比較，也不會上傳或自動備份。

介面針對 iPad mini 直向與橫向調整，四個主入口固定在同一列或穩定的四欄中，不使用水平捲動；主要觸控按鈕至少 44 CSS px，並支援觸控裝置較大的操作區。「柔和」中文／英文 rate 為 0.76／0.75，「再慢一點」為 0.70／0.69；相鄰短句會合成一次自然朗讀，較長內容才分段，快速重播會取消上一段，避免聲音重疊。中文會先找裝置上的台灣中文 enhanced／natural voice，英文優先美式自然音色，再使用其他相容 voice；家長可逐一試聽並固定選擇。若換裝置後找不到原音色，網站會安全改回自動最佳音色。實際聲線仍取決於作業系統已安裝的 voice，網站不宣稱等同真人錄音。

## 使用方式

直接開啟 `index.html`，或使用 GitHub Pages 網址。建議使用 iPad mini 或較大的觸控畫面，也可在 Safari、Chrome 或 Edge 的近期版本使用；第一次可到「今天 → 家長小站 → 兔兔聲音試聽室」比較這台裝置實際可用的音色。

長期內容規劃與分齡解鎖原則請見 [`CURRICULUM_ROADMAP.md`](CURRICULUM_ROADMAP.md)。
